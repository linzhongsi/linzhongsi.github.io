定义

内存布局

直接赋值： String s = “a” 常量池 有就直接指向 没有则创建新的 最后是**常量池的空间地址 **

调用构造器： String s = new(“a”) 堆中创建空间，维护vlaue属性，value指向常量池的空间。最后s是指向堆中的空间地址

每次更新都需要开辟新空间

不可变

String 是 final类 保存串的数组是final修饰的 存放于字符串常量池 数据结构为HashTable

所以 一旦分配了 内容不可变 如果改变了就是指向了新的对象

常见方法

子类

StringBuffer

可变的字符串序列 可以对内容增删 所以长度也是可变的 容器

final类 不可被继承

内存在char[] value中 不需要变更地址

更高的效率（因为可以直接修改内容 而不是创建新对象并指向）

转化

String - StringBuffer : new StringBuffer(s1)

StringBuffer - String : 1 new String(s1) 2 .toString()

StringBuilder

更高效，所以线程不安全 建议单线程
